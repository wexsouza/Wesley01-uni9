Complemento: Estrutura HTML Essencial

Reforçando: O Esqueleto da Página

Todo documento HTML começa e termina com as mesmas linhas. Elas dizem ao navegador o que ele está lendo. Memorize esta estrutura!

<!DOCTYPE html>
<html lang="pt">
<head>
   <meta charset="UTF-8">
   <title>Título da Página</title>
</head>
<body>
   <!-- Seu Conteúdo Aqui -->
</body>
</html>

Tags de Texto Simples e Semântica

Usar a tag correta não só organiza seu código, mas também ajuda o Google (SEO) e pessoas com deficiência visual (Acessibilidade).

  • <p>: Use para parágrafos de texto comum.
  • <h1> a <h6>: Use apenas para títulos e subtítulos. **Nunca pule níveis** (Ex: Não vá de `

    ` para `

    `).

  • <strong>: Não use apenas para negrito. Use para dar **importância** a uma palavra.
  • <em>: Não use apenas para itálico. Use para dar ênfase (o navegador "lê" com entonação).

Links e Imagens (Os Elementos Chave)

Os links e as imagens são fundamentais para o HTML. Lembre-se sempre destes atributos:

Link: <a>

<a href="pagina.html" target="_blank">Clique Aqui</a>
  • `href`: O destino do link.
  • `target="_blank"`: Abre o link em uma nova aba do navegador.

Imagem: <img>

<img src="caminho/foto.jpg" alt="Descrição da Imagem">
  • `src`: O caminho (source) onde a imagem está salva.
  • `alt`: A **descrição obrigatória** da imagem. Ajuda a acessibilidade e o SEO.
